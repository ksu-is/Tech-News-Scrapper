### Project Roadmap: Tech News Scraper

### Core Features Implemented

- [x] **Multi-Website Scraping Capability**  
  Enable the program to scrape data from multiple news websites in a single run, allowing for broader coverage of technology news sources.

- [x] **Controlled Output Limiting**  
  Restrict output to a maximum of **10 headlines per news outlet** to improve readability and prevent information overload.

- [x] **Welcome Prompt Initialization**  
  Add an interactive welcome message that runs at startup and clearly guides the user into the program workflow.

- [x] **Modular Function Design**  
  Refactor the primary execution logic into a dedicated function to improve code organization, reusability, and maintainability.

- [x] **User-Selected News Sources**  
  Prompt the user to choose from a predefined list of available news outlets before scraping begins, ensuring customizable and targeted data collection.

- [x] **Clean Output Formatting (Menu Suppression)**  
  Prevent unnecessary menu or interface elements from being printed during scraping, ensuring only relevant results are displayed.

- [x] **Enhanced Output with Source Links**  
  Display each scraped headline alongside its corresponding URL, improving traceability and allowing direct access to the original article.
