# Tech News Scraper

## Developer
Daniel Gammon


## Overview
Tech News Scraper is a Python-based web scraping program designed to gather the latest technology news from selected online sources and present the results in a clean, structured layout directly within the terminal. When the program launches, it displays a welcome message and prompts the user to choose from a fixed list of trusted technology news websites. After the user selects their preferred sources, the software automatically scrapes the most recent article headlines from each site and organizes them by news source for easy reading. Each headline is paired with its corresponding article link, allowing users to quickly access full stories that interest them. By consolidating multiple technology news outlets into one streamlined interface, the program saves users time, reduces the need to browse several websites individually, and creates a more efficient way to stay informed on current developments in the tech industry.

## Features
- Interactive terminal welcome menu
- Allows users to choose from a preset list of news sources
- Scrapes current technology news articles from selected websites
- Displays results in a clean, organized terminal layout
- Uses article titles with direct clickable links to full articles
- Separates and organizes articles by news source
- Automates collecting news from multiple websites into one streamlined interface

## Purpose
This project helps users save time by automatically gathering technology news from multiple trusted sources and displaying it in one convenient, organized location. Instead of manually visiting several websites to search for updates, users can quickly view the latest headlines in a structured terminal interface and open only the articles that interest them. By streamlining the news discovery process, the program reduces clutter, improves efficiency, and provides a faster, more convenient way to stay informed on current trends and developments in the technology industry.
