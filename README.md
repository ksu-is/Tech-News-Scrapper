# Tech News Scraper

## Developer
Daniel Gammon


## Overview
Tech News Scraper is a Python-based web scraping program that gathers the latest technology news from selected online sources and exports the results into a structured CSV file. When the program runs, it displays a welcome message in the terminal and prompts the user to choose from a fixed list of available news websites. After the user selects their preferred sources, the software scrapes article titles and stores it in a CSV file, with article titles serving as clickable hyperlinks and results organized by their respective news source.

## Features
- Interactive terminal welcome menu
- Allows users to choose from a preset list of news sources
- Scrapes current technology news articles from selected websites
- Exports results into a CSV file
- Uses article titles as clickable hyperlinks
- Separates and organizes articles by news source
- Automates collecting news from multiple websites into one file

## Purpose
This project helps users save time by automatically gathering technology news from multiple trusted sources into one convenient and organized document.
